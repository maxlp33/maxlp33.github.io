Creative Inspiration

Netflix's Arcane animation is renowned for its unique blend of hand-painted style with 3D technology. The challenge of this project was recreating that expressive 2D art style in a 3D model while maintaining the character's iconic features and personality.

Project Overview

This project focused on recreating the unique visual style of Caitlyn character from Netflix's Arcane animation. I studied the show's art style in depth, particularly how it combines hand-painted textures with 3D technology, and developed a custom material system to mimic this effect.

Character modeling focused on expressive capabilities and animation-friendly topology. The material system combines hand-painted textures with procedural nodes, creating a visual style that's both modern and traditionally artistic. The entire project showcases innovative approaches to stylized 3D character design.

Artistic Process

The project began with in-depth analysis of Arcane's visual language, including color usage, lighting treatment, and material representation. I created multiple test renders to experiment with different shader settings, ultimately developing a material system capable of recreating the animation's texture quality. The character sculpting phase focused particularly on capturing Caitlyn's personality traits and facial details.
